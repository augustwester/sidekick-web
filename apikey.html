How to Get an API Key

In order to use Sidekick, you need to

  1. Enable billing on your OpenAI account
  2. Create a unique API key

There are no costs associated with this until you start using the app. You only pay for what you use, which (as of this writing) is $0.002 / 1K tokens. To put this into perspective, this means that you can exchange ~2500 pages of text for $2. This is much more than most people will use in the course of many months.

There are no hidden fees, and the developer of Sidekick does not profit in any way from use of the app.

1. Enable billing

  1. Go to your OpenAI account.
  2. In the upper right-hand corner, click Personal then Manage account
  3. In the left side of the screen, select Billing and choose Set up paid account
  4. Follow the instructions to enable billing

2. Create API key

  1. In the left side of the screen, select API keys
  2. Choose Create new secret key
  3. Enter the name of your key (e.g. “Sidekick”)
  4. Copy the key
  5. Find Sidekick in the menu bar (icon is ) and choose Settings...
  6. Click Add API Key... and paste in the key
  7. Click Add

Sidekick should now be working. Please note that your API key is secret and should never be shared with anyone.
